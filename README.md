# Railway Timetable Management System (C++)

## Overview

The Railway Timetable Management System is a simple console-based application developed using C++.
It is designed to manage train schedule information such as train number, arrival time, and departure time.

The system provides a menu-driven interface where users can easily view the train timetable and search for train details.

## Features

* Display the railway timetable
* Search train by train number
* Convert time from 24-hour format to 12-hour format
* Simple and easy-to-use console interface

## Technologies Used

* C++
* Standard C++ Libraries

## Objective

The aim of this project is to demonstrate how C++ programming can be used to build a simple system for organizing and managing train schedule information.
